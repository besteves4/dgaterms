Abstract

The Data Governance Act Vocabulary (dgaterms) is a vocabulary created to (1) generate machine-readable policies for the reuse of public data, (2) specify data altruism consent terms and (3) create uniform registries of data altruism organisations and intermediation servicesâ€™ providers as the usage of such technologies promotes machine readability and interoperability.
